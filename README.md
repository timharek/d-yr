# d-Yr

Using the Yr.no / MET.no weather API with Deno.

## Setup

To change the `coordinates`, update them in `App.ts`.

Run the project with:
```sh
# For full forcast
deno run --allow-net App.ts -f 
# OR for current
deno run --allow-net App.ts -c
```
